# sample_mlts

## Pré Requisitos
Faça download das chaves para uso do exemplo no site (https://badssl.com/download/) e salve o conteudo dentro da pasta keys na raiz do projeto

## Instalação
Install with npm:

```shell
$ npm install
```

## Uso

```shell
$ node index.js
```
